## Название проекта: 

Обзор метрик, оценка монетизации и юнит-экономики, поиск инсайтов и точек роста, A/B-тестирование маркетплейса.

## Описание проекта: 
Анализ данных по заказам, затратам на каналы привлечения, пользователям и транзакциям от быстрорастущего маркетплейса, 
который уже нашёл свою нишу и начал приносить прибыль с целью:

- оценки динамики основных продуктовых метрик (Revenue, ARPU, MAU, WAU, DAU, RR, CTR, CR в заказ),
- оценки метрик монетизации и юнит-экономики (AOV, LTV, CAC, ROI, Payback Period, Stickiness, динамика набора новых пользователей),
- поиска инсайтов и точек роста (анализ метрик по каналам, когортам, сегментам, категориям товаров, построение продуктовых воронок),
- исследования результатов A/B тестирования (ключевая, вспомогательная и барьерные метрики). 

Для анализа были использованы библиотеки pandas, matplotlib, numpy, seaborn, scipy (stats, ttest_ind), statsmodels (proportions_ztest).

## Общие выводы:

- Общее снижение выручки компании (Revenue) и конверсии (CR) при сохранении среднего дохода с пользователя (ARPU) и количества активных пользователей (DAU,WAU,MAU).
- Средний доход с пользователя в динамике достаточно стабилен при высокой волатильности средней стоимости заказа. При этом соотношение ценности клиента с затратами на него 
свидетельствует об окупаемости продукта, где LTV-клиента окупается в первую неделю после регистрации.
- Изменение объёмов притока пользователей по каналам привлечения. Снижение покупок по наиболее часто покупаемой и наиболее прибыльной категориям, снижение количества наиболее платящих пользователей. Отсутствие зависимости конверсии в заказа от типа устройства, операционной системы и региона пользователя. Воронка конверсии пути пользователя показывает потери на этапах оформления заказа.
- Результат A/B эксперимента подтвердил гипотезу о влиянии бонуса на увеличении конверсии для пользователей канала TikTok.
